# Code Of Conduct

This project should be a calm, useful place to collaborate.

Expected behavior:

- Be respectful and specific.
- Critique ideas, code, and trade-offs without attacking people.
- Assume good intent, but accept corrections when impact differs from intent.
- Keep issues and pull requests focused on the project.
- Avoid harassment, discriminatory language, sexualized language, threats, or sustained disruption.

Maintainers may edit, hide, lock, or remove comments and may limit participation when behavior makes collaboration unsafe or unproductive.

If you need to report a conduct problem, contact the maintainers privately when a private channel is available. If no private channel has been published yet, open a minimal issue asking for one without naming people or sharing sensitive details publicly.
