Changelog 更新履歴

Latest updates and release notes for IAI for Safari. IAI for Safari の最新のアップデートとリリースノート。

v1.0.0 (Initial Release)

2026-06-13

Vim-style Link Hints Vim風リンクヒント機能

  • Press "f" to toggle hint tags on links, buttons, and inputs, and type the characters to click. 「f」キーを押すことですべてのリンクやボタン、インプットにヒントタグを重ねて表示し、表示文字を入力して即座にクリック可能。
  • Staggered and non-overlapping hint placement algorithm prevents labels from covering vital text. 表示領域内の文字を覆い隠さないための、タグの重なりを防ぐスマートな配置アルゴリズムを搭載。
  • Full support for elements inside Shadow DOMs, solving generic extension coordinate positioning issues. Shadow DOM内の要素を検出し、一般的な拡張機能で座標取得がバグる問題を解消。

Spotlight-style Command Palette Spotlight風コマンドパレット

  • Press "o" to open a modern glassmorphic Spotlight-style search bar. 「o」キーで起動する、美しいガラスモーフィズム（Glassmorphism）のSpotlight風検索バー。
  • Search across open tabs, bookmarks, history, and extension shortcuts using bilingual (English and Japanese) search capabilities. 開いているタブ、ブックマーク、履歴、拡張機能コマンドを、英語と日本語のバイリンガルで横断検索可能。

Bilingual Options & Setup バイリンガル対応と詳細設定

  • Live options previewer in the options page to immediately see how customized hints and palettes look. 設定画面（Optionsページ）で、カスタマイズしたヒントやパレットの見た目をリアルタイムでプレビューできるライブプレビュー機能。
  • Custom UI language setting to easily switch the extension interface between English and Japanese. 拡張機能のインターフェース言語を、英語と日本語から選択できるカスタム言語切替設定。
  • Customizable hint keys, border styles, smooth dim transition speeds, and scroll indicators. ヒントキーのカスタム設定、ボーダースタイル、滑らかなディム遷移の速度設定、スクロールインジケータ表示。

IME Safety & Local-First Privacy IME誤動作防止と完全ローカル設計

  • IME safe key event routing to prevent shortcut triggers from swallowing active composition and confirmation keys. 日本語IMEでの文字入力・確定操作（Enterキー等）中にショートカットキーが暴発するのを防ぐ安全なイベントルーティング。
  • Strictly local-first design with zero analytics, trackers, telemetry, or remote servers. All your browsing details remain on your device. トラッカー、アナリティクス、外部通信を一切行わないローカルファーストの徹底。入力履歴や設定はすべてユーザー端末内でのみ処理されます。
