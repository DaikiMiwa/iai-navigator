Designed for Power Users ブラウジングを極限まで効率化

Packed with key capabilities to enhance your Safari productivity. Safariの生産性を向上させる、洗練された数々の機能を搭載しています。

f

Keyboard Link Hints リンクヒント表示

Press "f" to overlay hint tags on links, buttons, and form inputs. Type the matching letters to click instantly. 「f」キーを押すと、すべてのリンクやボタンにヒントタグをオーバーレイ表示。表示された文字を入力するだけで瞬時にクリックします。

o

Spotlight Command Palette コマンドパレット

Access open tabs, bookmarks, history, and core extension commands via a beautiful macOS-style Spotlight command bar. 「o」キーで起動。開いているタブ、ブックマーク、履歴、さらには拡張機能の操作コマンドをSpotlight風バーから横断検索できます。

Local-First Privacy 徹底したプライバシー保護

Zero analytics, trackers, telemetry, or remote servers. All your browsing, search queries, and settings are kept strictly local. トラッカー、アナリティクス、外部サーバーへの通信は一切ありません。入力内容や履歴、設定データは完全にユーザー端末内でのみ処理されます。

あ

Japanese IME Safe 日本語IMEへの完全対応

Smart keyboard event routing prevents shortcut triggers from swallowing your active IME composition and confirmation keys. IME入力中の文字確定操作やEnterキー押下により、意図せずキーボードショートカットやコマンドが暴発するのを防ぎます。

Easy Setup セットアップ手順

  1. Compile the Xcode project or install the host application from the Mac App Store. Xcodeからビルドを実行するか、Mac App Storeからアプリをインストールします。
  2. Launch the app, then click the button to open Safari Settings. アプリ本体を起動し、画面内のボタンをクリックしてSafariの設定を開きます。
  3. Navigate to Extensions, check the box next to Safari Keyboard Navigation, and set permission to "Always Allow on Every Website". 拡張機能タブを選択し、Safari Keyboard Navigationのチェックを入れて「常にすべてのWebサイトで閲覧を許可」を選択します。
  1. Install the app from the App Store on your iPhone or iPad. iPhoneまたはiPadのApp Storeからアプリをインストールします。
  2. Open device Settings > Safari > Extensions. 端末の設定アプリを開き、Safari > 拡張機能の順に進みます。
  3. Tap Safari Keyboard Navigation, toggle it on, and grant website access permissions. Safari Keyboard Navigationをタップして有効化し、Webサイトへのアクセス権限を許可します。
